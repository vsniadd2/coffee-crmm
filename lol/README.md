# Импорт данных в БД

**Как запустить скрипты заполнения БД:**

1. Убедитесь, что БД запущена (например: `docker compose up -d` или локальный PostgreSQL).
2. Из **корня проекта** выполните:
   ```bash
   npm run import-data
   ```
   или напрямую:
   ```bash
   node lol/import-all.js
   ```
   Запускаются по очереди: импорт из CSV, затем Silver, затем Gold. Для Silver/Gold в корне должен быть установлен пакет `xlsx` (`npm install` в корне проекта).
