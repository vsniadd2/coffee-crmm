# Миграции БД

Миграции выполняются при старте приложения (без пересоздания Docker-контейнера).
Файлы в папке `migrations/` выполняются по порядку (по имени); каждая применяется один раз.
Состояние хранится в таблице `schema_migrations`.

Формат имени файла: `NNN_описание.sql` (например `001_points_system.sql`).
